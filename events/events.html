Question 1: What is HTML?
HTML stands for Hypertext Markup Language, used to structure web pages.
Question 2: What is CSS?
CSS stands for Cascading Style Sheets, used to style HTML elements.
Question 3: What is JavaScript?
JavaScript is a programming language used to create interactive effects within web browsers.
Question 4: What is a DOM?
DOM stands for Document Object Model, which represents the page structure as objects.
Question 5: What is a browser?
A browser is a software application used to access and view websites.
